# Claude Code — Instrucciones del proyecto

## Acción obligatoria al inicio de cada sesión

Antes de hacer cualquier cosa, lee siempre:

1. `AGENTS.md` — convenciones, skills y forma de trabajar en este proyecto
2. `../project.spec.md` — descripción completa del monorepo, principios, stack y estado global (vive en la raíz del monorepo)

## Metodología

Este proyecto usa **Spec-Driven Development (SDD)**. Cada cambio relevante tiene su propio spec co-localizado con el código. No implementes nada sin spec previo salvo que el usuario lo indique explícitamente.

## Idioma

Responde siempre en **castellano de España**.
